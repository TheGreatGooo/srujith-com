# srujith.com

Backend developer & tinkerer portfolio. Terminal-inspired, utilitarian, dark.

## Tech Stack

- **Vite + React** — frontend framework
- **Cloudflare Pages** — hosting
- **JetBrains Mono + Inter** — fonts

## Running Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content

Real projects from GitHub and Gitea repos across 7 categories:
- Backend & Infrastructure
- Email & Privacy
- Home Lab & IoT
- Crypto & Trading
- AI & LLM
- Networking & Systems
- Media & Music

## Design

Terminal boot sequence on page load. No fluff — just projects, descriptions, and links. Dark terminal aesthetic with monospace fonts, green/amber accents.
